To use bower on existing project do the following

http://git-scm.com/book/en/v2/Git-Basics-Getting-a-Git-Repository

1. Install Nodejs (64bit)
2. open windows powershell
3. type => npm install -g bower   --- this will install bower using node package manager
4. go to local project directory
5. type => git init
6. then type => bower install  --- this will download all the libaries in "bower_components" folder
